# Plan testów projektu

## 1. Cel testów

Celem testów jest sprawdzenie poprawności działania kluczowych elementów systemu rekomendacji produktów:

- przetwarzania zapytań tekstowych,
- rozpoznawania ceny,
- obsługi synonimów,
- obliczania podobieństwa TF-IDF,
- scoringu rekomendacji,
- działania interfejsu użytkownika,
- działania REST API.
